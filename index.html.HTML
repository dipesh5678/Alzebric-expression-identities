11 Identites

1. a2-b2=(a+b)(a-b)


EXAMPLE

2. (a+b)2= a2+2ab+b2


EXAMPLE

3. (a-b)2= a2-2ab+b2


EXAMPLE
(2-4)2=22-2*2*4+42

4. (a+b)3=a3+b3+3ab(a+b)


EXAMPLE
(2+4)3=23+43+3*2*4(2+4)

5. (a-b)3=a3-b3-3ab(a-b)


EXAMPLE
(2-4)3=23-43-3*2*4(2-4)

EXAMPLE

7. a3-b3=(a-b)a2+ab+b2


EXAMPLE

EXAMPLE
